# ⚖️ Fair Hiring AI

AI-powered hiring system that eliminates gender bias using Fairlearn & explains decisions with SHAP.

## 🚀 Features
- Bias Dashboard (before vs after fairness)
- Single Candidate Evaluator with confidence score
- SHAP Explainability Graph
- Batch CSV Upload & Gender Comparison Chart
- PDF Report Download

## 🛠️ Tech Stack
Python | Streamlit | Scikit-learn | Fairlearn | SHAP | FPDF2

## ▶️ How to Run
```bash
pip install streamlit pandas scikit-learn fairlearn shap matplotlib fpdf2
python generate_data.py
streamlit run app.py
```

- Bias reduced by 77% using Fairlearn
